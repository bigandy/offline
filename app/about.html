Colophon

This site is powered by WordPress on Nginx. I have secured everything with ssl/tls and it utilises http/2 and it is https://big-andy.co.uk.

The theme is built on top of the Zurb Foundation 5 Grid. Code linting and concatenation and minification, autoprefixing, uncss, criticalcss, svg optimisation is all handled by Gulp and CSS is compiled by PostCSS.

For the typography I am using Open Sans from Google Fonts via the Google Webfont Helper (I can self-host giving a 100/100 pagespeed insights score)… combined with localFont I can inline the font in base-64 format in font.css and then cache in the browser’s localStorage.

All my code is on Github.
